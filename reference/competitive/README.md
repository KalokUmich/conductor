Ongoing study of competing products. Updated monthly alongside Claude Code reference analysis (Phase 9.1-9.9).

## Products

| Product | Type | Source | Our Advantage | Their Advantage |
|---------|------|--------|---------------|-----------------|

## Process

1. Monthly: review changelogs + new features (2-4h per product)
2. Document: structured notes in `{product}/YYYY-MM.md`
3. Extract: create actionable items under relevant Phases if a pattern is worth adopting
4. Track: update comparison matrix above
